# Smart Campus Companion

A mobile application designed to improve campus communication, scheduling, navigation, and student access to university information.

## Project Goals
- Centralized announcements
- Real-time schedule updates
- Campus navigation
- Staff directory
- Academic calendar integration

## Tech Stack
- Flutter
- Firebase / Supabase
- GitHub

- UI/UX
- Frontend Development
